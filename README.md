# Mishell Rose Mathew — Portfolio

Dark single-page portfolio built with React, Vite, TypeScript, Tailwind CSS, GSAP, Framer Motion, and hls.js.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Content lives in `src/lib/data.ts` (projects, experience, skills, contact).
